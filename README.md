
# 🚀 GameLift

GameLift is a modern Windows application designed to improve your gaming experience with profile management, settings, and community features.

## ✨ Features

- 🎮 Modern user interface
- ⚙️ Profile and settings management
- 🌐 Community features
- 🔄 Regular updates and improvements

- .NET 8 Desktop Runtime
